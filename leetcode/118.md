
定一个非负整数 numRows，生成杨辉三角的前 numRows 行。

在杨辉三角中，每个数是它左上方和右上方的数的和。

示例:

输入: 5

输出:

[
     [1],
     
    [1,1],
    
   [1,2,1],
   
  [1,3,3,1],
  
 [1,4,6,4,1]
 
]

```

/**
 * @param {number} numRows
 * @return {number[][]}
 */
var generate = function(numRows) {
    let res = []
    for (var i = 0; i < numRows; i++) {
        res[i] = []
        res[i][0] = 1
        if (i === 0) continue
        for (var j = 1; j < i; j++) {
            res[i].push(res[i-1][j] + res[i-1][j-1])
        }
        res[i].push(1)
    }   
    return res
};
```
